# clinica-spring-security
### Implementação da camada de segurança incluindo:
- Página de Login
- Autenticação e autorização
- Bloqueio de páginas por tipo de usuário

### Arquitetura
- Padrão Model-View-Controller (MVC)
 
### Backend
- Java Spring Boot
- Spring security
- Spring Data
- JPA Hibernate
- Bean Validation
 
### Banco de dados
- MySql

### FrontEnd
- Thymeleaf e BootStrap 4
